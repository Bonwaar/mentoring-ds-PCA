Principal Component Analysis (PCA)

A Simple Method to Understand Your Data

What is PCA?

PCA is a technique that helps simplify complex data

üéØ Main Goal:

Transform many correlated variables into a few independent variables called "principal components"

Before PCA

10 variables hard to visualize and understand

After PCA

2-3 components that summarize the essentials

How Does It Work? (Simplified)

Step 1: Standardization

Put all variables on the same scale (mean = 0, standard deviation = 1)

Step 2: Finding Directions

Search for directions where data varies the most

Like finding the best camera angle for a photo

Step 3: Creating Components

Each component is a combination of original variables

PC1 = 0.3√óVar1 + 0.5√óVar2 + 0.2√óVar3 + ...

Step 4: Selection

Keep only components that explain the most variance
